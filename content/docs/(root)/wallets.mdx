Trezo handles wallet connectivity through each network package's modal provider. The wallets available to your users depend on which package you're using and which modal you configure.

---

## By Network Package

Each Trezo package manages wallet connectivity for its blockchain. The wallets and modal providers available differ per package.

<Steps>

### @trezo/evm

Supported modal providers configured via `modalConfig.from`:

| Provider                                | Description                                                      |
| --------------------------------------- | ---------------------------------------------------------------- |
| [ConnectKit](https://family.co/docs)    | Polished connect UI for MetaMask, Coinbase, and injected wallets |
| [Reown AppKit](https://docs.reown.com/) | 300+ wallets via WalletConnect, MetaMask, Coinbase               |

See the [documentation](/docs/packages/evm) for setup details.

</Steps>
