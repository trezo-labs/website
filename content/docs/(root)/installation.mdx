## Quick Start

Create a new Trezo project using the CLI command:

```bash
npx trezo@latest init
```

See the [Trezo CLI](/docs/cli) documentation for more information.

## Select a Network Package

Each network package provides a type-safe integration layer, wallet connectivity, and developer-friendly APIs for interacting with that blockchain.
