### Quick Start

Run the command install Trezo and initialize a new project:

```bash
npx trezo init
```

For the full details and updated flow, see the [CLI documentation](/docs/cli).

---

### Core Packages

Install only the package for the blockchain you're building on. Each package bundles its own dependencies, so a single install is all you need.
