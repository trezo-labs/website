## Overview

Trezo CLI simplifies Web3 project setup. It handles; **Network Package selection**, **Project templates**, **Post-setup commands**

---

<Steps>

### Initialize Project

```bash
npx trezo@latest init
```

You can also specify your project name after the `init` command.

---

### Post-Setup Commands

Some templates include additional setup commands:

```bash
◇  Optional post-setup commands: ─╮
│                                 │
│  1. cd trezo-app                │
│  2. pnpm install                │
│                                 │
├─────────────────────────────────╯
│
◇  Run post-setup commands?
|  ● Yes / ○ No
```

You can run them automatically from the CLI or skip and run them manually later

</Steps>

---

## Generated Project

After setup, your project includes:

- Pre-configured Trezo integration
- Selected blockchain package installed
- Wallet connection support
- Project structure based on your template

Your project is ready for development — a fully working Web3 app out of the box.
