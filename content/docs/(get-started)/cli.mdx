## Overview

Trezo CLI simplifies Web3 project setup. It handles:

- Network Package selection
- Project templates
- Post-setup commands

This saves you from manually configuring SDKs, dependencies, and project structure.

---

<Steps>

### Initialize Project

Run the CLI:

```bash
npx trezo@0.1.6 init
```

> Use version 0.1.6; we're actively working on fixing the current, which is unstable.

You can also specify a project name after the `init` command.

### Global Installation

You can also install the CLI globally:

```bash
npm install -g trezo
```

After installing, you can run `trezo init` directly without using npx.

> This will launch an interactive setup to guide you through project creation.

### Post-Setup Commands

Some templates include additional setup commands:

```terminal
The following post-setup commands will complete your project setup:

1. cd trezo-project
2. pnpm install

? Would you like to run these commands now? (Y/n)
```

- You can run them automatically from the CLI
- If skipped, run them manually later

## Generated Project

After setup, your project includes:

- Pre-configured Trezo integration
- Selected blockchain package installed
- Wallet connection support
- Project structure based on your template

Your project is ready for development — a fully working Web3 app out of the box.
