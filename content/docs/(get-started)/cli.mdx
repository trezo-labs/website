## Overview

Trezo CLI simplifies Web3 project setup. It handles:

- Blockchain package selection
- Project templates
- Directory conflicts (overwrite, rename, cancel)
- Optional post-setup commands

This saves you from manually configuring SDKs, dependencies, and project structure.

---

<Steps>

### Initialize Project

Start by running:

```bash
npx trezo init
```

This launches an interactive setup that will guide you step by step.

### Handling Existing Directories

If a directory with the chosen project name already exists:

```terminal
? A directory named "trezo-project" already exists. What would you like to do?
> Overwrite existing project
  Rename project
  Cancel operation
```

- **Overwrite** – deletes the existing folder and continues
- **Rename** – prompts for a new project name and validates it
- **Cancel** – exits the CLI safely

> Pressing CTRL+C at any point will cancel the operation.

### Post-Setup Commands

Some templates include additional setup commands:

```terminal
The following post-setup commands will complete your project setup:

1. cd trezo-project
2. pnpm install

? Would you like to run these commands now? (Y/n)
```

- You can run them automatically from the CLI
- If skipped, run them manually later
