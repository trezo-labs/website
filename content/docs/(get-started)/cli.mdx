## Overview

The CLI fetches curated Trezo templates from a remote registry and clones them directly into your local environment

---

## Initialize Project

```bash
npx trezo init
```

Run the CLI directly **without installation** using your package manager of choice

---

## Commands

<Steps>

### **trezo init**

Initialize a new multi‑chain Web3 project.

```bash
trezo init [options]
```

#### Options

- `-n, --name <name>`: Project name / directory name. If omitted, you will be prompted.
- `-p, --package <name>`: Skip interactive package selection and use the specified package ID.
- `-t, --template <name>`: Skip interactive template selection and use the specified template ID.

#### Interactive Flow

When you run `trezo init` without flags, the CLI will guide you through:

1. **Package selection** – Choose from available protocol packages.
2. **Template selection** – Pick a framework or starter.
3. **Project directory** – Confirm or change the target folder name.
4. **Template download** – The selected GitHub template is cloned locally.

#### Example Usage

```bash
# Fully interactive
trezo init

# Skip prompts with flags
trezo init -n my-dapp -p @trezo/evm -t vite-app

# Full usage
trezo init --name my-dapp --package @trezo/evm --template vite-app
```

#### Post‑Setup Instructions

After scaffolding, the CLI displays any recommended post‑setup commands (e.g., `pnpm install`) defined by the template. You can copy and run them manually.

---

### **trezo list**

Display all available packages and their templates in a readable tree format.

```bash
npx trezo list
```

#### Example Output

```txt
📦 @trezo/evm – Ethereum { package: evm }
  ├─ Next.js → evm-next-js
  └─ Vite → evm-vite

📦 @trezo/strk – Starknet { package: strk }
  └─ Starknet React → strk-react

Found 2 package(s)

Usage: `trezo init -p <package> -t <template>`
```

Use this command to browse available scaffolds before running `trezo init`.

</Steps>

- Wallet connection support

Your project is ready for development — a fully working Web3 app out of the box.
